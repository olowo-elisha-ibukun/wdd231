WDD 231 – W05 Project: Site Plan

Site Name

Professional Development Hub

This name was chosen because it represents an organization dedicated to supporting professionals and their career growth by providing networking opportunities, resources, and development programs.

Site Purpose

The purpose of this website is to empower professionals by providing opportunities for networking, access to professional resources, mentorship programs, and career development information. The site serves as a hub for professionals to connect, learn, and advance their careers.

Scenarios

  • How can professionals register or join the Professional Development Hub community?
  • Where can visitors find upcoming networking events and professional development opportunities?
  • How can professionals contribute resources and share their expertise with the community?

Color Schema

Primary Color: Professional Blue (#0066cc): Used for the header, buttons, links, and major headings to create a professional and trustworthy appearance.

Secondary Color: Bright Teal (#00a6d6): Used for accents, taglines, and hover effects to add visual interest.

Dark Text: Dark Gray (#333333): Used for body text and main content for optimal readability.

Neutral Colors: Light Gray (#f8f9fa), White (#ffffff), and Dark Footer (#2c3e50): Used for page backgrounds, cards, and footer to improve overall readability.

Typography

Poppins is used as the primary font for the website with Inter as a secondary font. Headings use a bold weight to create hierarchy and emphasis, while body text uses a regular weight to ensure clarity and readability across all sections, including navigation, cards, forms, and footer.

Wireframe

Mobile View

  • Logo with Organization Name
  • Mobile Menu Toggle Button
  • Hero Section
  • Featured Opportunities Cards (Stacked)
  • About Section
  • Footer with Contact Info and Social Links

Tablet View

  • Logo and Organization Name | Navigation Links
  • Hero Section
  • Featured Opportunities Cards (2 columns)
  • About Section
  • Footer with 2-3 Column Layout

Desktop View

  • Logo | Organization Name | Full Navigation Menu
  • Hero Section with Call-to-Action
  • Featured Opportunities Cards (3+ columns)
  • About Section
  • Footer with 3 Column Layout (Contact, Social, Info)

HTML and CSS

This site plan is created using valid HTML and CSS, includes a proper page title, responsive design for mobile and desktop, and follows web standards for structure and styling. The site includes semantic HTML elements and professional styling.
